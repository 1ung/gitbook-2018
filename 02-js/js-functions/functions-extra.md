## Functions:

### 1.1 Functions basics
1.1.1
  - create a function that prints out 'Hello World'
  - call that function
  
Example:
```
function example() {
  \\ your code goes here
};

example();
```
or:

```
var example = function() {
  \\ your code goes here
};

example();
```
